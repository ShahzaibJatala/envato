🎨 Envato Landing Page
This project is a visually precise landing page clone of the Envato website. It marks a significant milestone as my first website built entirely with core web technologies. The focus was on mastering semantic structure, CSS positioning, and clean design implementation.

Deployment
When you're ready to view this project live or share it with others, you can deploy it easily. Since this is a static site (HTML/CSS), it runs efficiently on any web hosting platform.

To host this project for free using GitHub Pages, follow these simple steps:

Go to the Settings tab of your GitHub repository.

Navigate to the Pages section in the left sidebar.

Under "Build and deployment", set the source to Deploy from a branch.

Select the main branch and the /(root) folder, then click Save.

Within a few minutes, your site will be live at https://ShahzaibJatala.github.io/envato/.
